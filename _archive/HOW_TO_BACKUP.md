# How to Create a Backup (Windows)

## Simple Method - Copy the Folder

### Step 1: Open File Explorer
- Press **Windows Key + E** on your keyboard
- OR click the folder icon on your taskbar

### Step 2: Navigate to Your Project
1. In the address bar at the top, type or paste:
   ```
   C:\Users\dusti
   ```
2. Press Enter
3. You should see your `estimator-react` folder

### Step 3: Copy the Folder
1. **Right-click** on the `estimator-react` folder
2. Click **Copy** (or press Ctrl+C)

### Step 4: Paste as Backup
1. **Right-click** in an empty space in the same window
2. Click **Paste** (or press Ctrl+V)
3. Windows will create a copy named `estimator-react - Copy`

### Step 5: Rename the Backup (Optional but Recommended)
1. **Right-click** on `estimator-react - Copy`
2. Click **Rename**
3. Name it something like:
   ```
   estimator-react-backup-dec-30-2024
   ```
4. Press Enter

## ✅ You're Done!

You now have a backup. If anything goes wrong, you can:
1. Delete the `estimator-react` folder
2. Rename `estimator-react-backup-dec-30-2024` back to `estimator-react`
3. Everything will be exactly as it was

## Alternative: Use Desktop for Backup

If you want the backup somewhere easier to find:

1. Follow Steps 1-3 above to copy the folder
2. Click on **Desktop** in the left sidebar of File Explorer
3. **Right-click** in an empty space
4. Click **Paste**
5. The backup will be on your Desktop!

---

**Note:** This backup is ONLY your code files. Your database is in Supabase's cloud, so it's already backed up there automatically.
