# Phase 3: Install Dependencies

Before we can build the PDF viewer and measurement tools, we need to install some packages.

## Required Packages

Run this command in your terminal:

```bash
npm install react-pdf fabric
```

### What These Do:

**react-pdf** - Renders PDF files in React
- Displays PDFs page by page
- Handles zoom and navigation
- Works with our uploaded PDF plans

**fabric** - HTML5 Canvas library
- Powers the measurement tools
- Handles drawing on top of PDFs
- Manages shapes (lines, areas, markers)
- Supports zoom/pan interactions
